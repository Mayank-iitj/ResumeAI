## Project Information

This Resume Analyzer Pro was created by MAYANK SHARMA as a comprehensive, production-ready solution for:
- ATS (Applicant Tracking System) resume scoring
- Multi-format resume parsing (PDF, DOCX, TXT)
- Candidate ranking and comparison
- Resume optimization with AI-powered feedback
- Interactive web interface using Streamlit

### Development

- **Project Start:** December 2025
- **Status:** Production-Ready
- **License:** MIT License

## Technology Stack

Created using:
- Python 3.8+
- Streamlit for web interface
- scikit-learn for ML models
- sentence-transformers for semantic analysis
- pdfplumber, python-docx for parsing
- plotly for interactive visualizations
